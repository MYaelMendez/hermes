# Mesh Identity

Every `hermes_cli` surface in `æ` returns a `surface` dict. Canonical shapes:

- `kind`: bounded_private_client_mesh / view / private_client / ae_engineering_hub / media / ...
- `address`: scannable URI when available
- `runtime`: hermes-agent / ffmpeg / ...
- `governance`: required token + audit tracer for sovereign surfaces

Bare status queries are the metastability hazard. Scalar supremacy blocks bare
scalar tails before dispatchers mutate state.

Enforced by `_enforce_promote_scalar_supremacy()` in `conductor.py`.
