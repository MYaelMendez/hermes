## Install

```bash

```

## Use

Invoke the skill:

```bash
/skill æ
```

Then run commands in Hermes chat / desktop / gateway / web surfaces:

```
>_æ:#startabusiness→     generate agentic.html sovereign business page
>_æ: pc:// · pc://view · pc://dispatch default · +æ://identity · mcp://tools · vscode:// · qr pc://view <addr>
```

## Private Client Mesh

Each Hermes instance running this skill IS a private client mesh node.

- **Surfaces:** conductor dispatch + sovereign `æ.html` viewport + Monaco editor
- **Address:** `pc://mesh/user/<user_id>/<id>` per-user, `pc://mesh/global/<id>` shared
- **Runtime:** local-first, sovereign, cloud-free by default
- **Scalability:** composable with other skills/agents; no monolith required

## Sovereign Viewport

Open the agentic engineering hub:

```bash
hermes viewport open
```

Or directly open the sovereign shell if mounted:

```
pc://view http://127.0.0.1:5173
```

QR-addressable: `qr <addr>` from the viewport command palette.
